Module Parsing

This module is a functor providing a saner interface to your parser.

module type PARSER = sig ... end
module type S = sig ... end
module Make : functor (P : PARSER) -> S with type token = P.token and type result = P.result

Pass your parser/lexer to this functor to obtain a nicer interface, without having to write boilerplate code.
